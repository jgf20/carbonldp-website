Linked Data Platform Concepts

Carbon LDP is a standards-compliant Linked Data Platform (LDP), which allows you to work with RDF and to link data using HTTP and familiar web development tools and techniques. While Carbon provides much more than the W3C recommends, this document provides an introduction to the standard Linked Data Platform concepts.
